# Создание категории "Все темы"

Эта категория показывает все темы из всех категорий форума в одном месте.

## Автоматическое создание

Категория "Все темы" будет автоматически отображаться на главной странице, даже если она не существует в базе данных (как виртуальная категория).

## Создание в базе данных (опционально)

Если вы хотите создать реальную категорию "Все темы" в базе данных, выполните SQL скрипт:

```bash
# Для PostgreSQL
psql -U postgres -d forum_db -f create-all-topics-category.sql
```

Или выполните SQL вручную:

```sql
INSERT INTO categories (name, description, created_at)
SELECT 'Все темы', 'Все темы форума', CURRENT_TIMESTAMP
WHERE NOT EXISTS (SELECT 1 FROM categories WHERE name = 'Все темы');
```

## Особенности

- Категория "Все темы" показывает все темы из всех категорий
- В этой категории нельзя создавать новые темы (форма создания скрыта)
- Каждая тема отображается с указанием её реальной категории в квадратных скобках
- Категория отображается первой в списке категорий на главной странице
- Кнопка удаления категории скрыта для "Все темы"

## Примечание

Если категория "Все темы" существует в базе данных, она будет использоваться с реальным ID. Если её нет, будет использоваться виртуальная категория с ID 'all-topics'.
