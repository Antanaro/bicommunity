ИНСТРУКЦИЯ ПО СОЗДАНИЮ ФАЙЛА .env
=====================================

1. В папке backend вы увидите файл "env.txt"
2. Откройте файл "env.txt" в блокноте
3. Измените пароль в строке DB_PASSWORD=postgres на ваш реальный пароль PostgreSQL
4. Сохраните файл
5. Переименуйте файл "env.txt" в ".env" (уберите расширение .txt, оставьте только точку в начале)

ВАЖНО: 
- Файл .env может быть скрыт в проводнике Windows
- Чтобы увидеть скрытые файлы: Вид → Показать → Скрытые элементы
- Или используйте редактор кода (VS Code, Cursor) - там файл будет виден

АЛЬТЕРНАТИВНЫЙ СПОСОБ:
1. Откройте PowerShell в папке backend
2. Выполните: Rename-Item env.txt .env
3. Файл будет создан и переименован
