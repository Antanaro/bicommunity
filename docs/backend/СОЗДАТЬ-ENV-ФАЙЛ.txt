ИНСТРУКЦИЯ ПО СОЗДАНИЮ ФАЙЛА .env
=====================================

ВАЖНО: Файл .env должен быть в КОРНЕ проекта (не в backend/)

1. В корне проекта скопируйте файл env.template в .env:
   ```bash
   cp env.template .env
   ```
   
   Или в Windows:
   - Скопируйте env.template
   - Переименуйте копию в .env

2. Откройте файл .env в блокноте или редакторе кода

3. Заполните необходимые значения:
   - DB_PASSWORD - пароль PostgreSQL
   - JWT_SECRET - сгенерируйте: openssl rand -hex 32
   - Остальные переменные по необходимости

ВАЖНО: 
- Файл .env может быть скрыт в проводнике Windows
- Чтобы увидеть скрытые файлы: Вид → Показать → Скрытые элементы
- Или используйте редактор кода (VS Code, Cursor) - там файл будет виден
- Все сервисы (backend, frontend, docker) используют один .env в корне
